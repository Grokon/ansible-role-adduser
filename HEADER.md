## Example Playbook

```yaml
- hosts: all
  roles:
    - grokon.adduser
  vars:
    adduser__groups:
      - name: "foo"
        gid: 3001
        sudo: /usr/bin/sudo
    adduser__users:
      - name: "boo"
        groups: ["foo"]
        ssh_key: ["ssh-rsa blahblahblah"]
        shell: "/bin/bash"
        uid: 3000
        home: "/home/foo"
        system: true
```
